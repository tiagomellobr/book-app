# Books APP

### Run Project

[Install Symfony CLI](https://symfony.com/download)

```bash
$ symfony composer install
```

```bash
$ docker-compose up -d
```

```bash
$ symfony console doctrine:migrations:migrate
```

```bash
$ symfony console doctrine:fixtures:load 
```

```bash
$ npm i
```

```bash
$ npm run dev
```

```bash
$ symfony server:start -d
```
